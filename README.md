# 🛡️ Sistema de Gestão - RX

> Uma aplicação fullstack completa para automatizar, gerenciar e centralizar as operações de uma facção no GTA Roleplay.

## 💻 Sobre o Projeto
Este projeto nasceu da necessidade de superar as limitações de gestão exclusivas via Discord. O sistema oferece uma interface web intuitiva para controle de membros, gestão financeira, controle de estoque (baú), planejamento de ações táticas e recrutamento, tudo isso mantendo integração ativa com o servidor do Discord (Webhooks e Bot).

Desenvolvido como projeto prático para o portfólio de Análise e Desenvolvimento de Sistemas (ADS).

## ✨ Principais Funcionalidades

* **Gestão de Membros:** Cadastro, controle de atividades (com upload de prints) e sistema de advertências com alteração automática de cargo no Discord.
* **Controle de Baú e Finanças:** Registro de depósitos, retiradas, controle de pendências (empréstimos de itens) e rateio automático de lucros pós-ações.
* **Planejamento de Ações (Elite):** Formulário interativo para escalação de membros em ações de tiro/fuga.
* **Comunicação Automatizada:** Abas de recrutamento, vendas e avisos da gerência integradas diretamente a canais do Discord via Webhooks.

## 🛠️ Tecnologias Utilizadas

O projeto adota uma arquitetura Fullstack utilizando as seguintes tecnologias:

**Frontend**
* React.js (Vite)
* Css

**Backend**
* Node.js com Express.js
* Integração com API do Discord (Webhooks e discord.js)

**Banco de Dados**
* PostgreSQL
